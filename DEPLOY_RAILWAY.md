# 🚂 RAILWAY DEPLOYMENT GUIDE - @kor_export Bot

## Что такое Railway?
Railway — это платформа для хостинга приложений. Позволяет запустить бота 24/7 бесплатно (с ограничениями).

## 📋 ПРЕДВАРИТЕЛЬНЫЕ ТРЕБОВАНИЯ
1. Аккаунт на [Railway.app](https://railway.app) (бесплатный)
2. GitHub аккаунт для загрузки кода
3. Telegram Bot Token (@BotFather)

## 🚀 ПОШАГОВАЯ ИНСТРУКЦИЯ DEPLOY

### ШАГ 1: Подготовка файлов на GitHub

1. Создай новый репозиторий на GitHub:
   - Название: `korea-exp-bot` или `kor-export-bot`
   - Visibility: Private (или Public)

2. Загрузи файлы в репозиторий:
   - bot.py
   - keep_alive.py
   - requirements.txt
   - settings.json
   - index.html (если нужен веб-интерфейс)
   - manifest.json

### ШАГ 2: Деплой на Railway

1. Зайди на [railway.app](https://railway.app)
2. Нажми **"New Project"** → **"Deploy from GitHub repo"**
3. Подключи свой GitHub аккаунт
4. Выбери репозиторий с ботом

### ШАГ 3: Настройка переменных окружения

В Railway Dashboard добавь переменные:

```env
BOT_TOKEN=8784793542:AAEDkX1igyp-MyWcqE4UYzBHHkymkDqrxqk
PYTHONIOENCODING=utf-8
```

**Где взять BOT_TOKEN:**
- Открой @BotFather в Telegram
- Напиши `/mybots`
- Выбери @korea_expbot
- Скопируй Token

### ШАГ 4: Настройка Start Command

В Railway в разделе **"Settings"** → **"Start Command** настрой:

```
python keep_alive.py
```

ИЛИ если используешь run_bot.py:

```
python run_bot.py
```

### ШАГ 5: Deploy

1. Нажми **"Deploy"** 
2. Подожди 2-3 минуты
3. Бот запустится автоматически!

## 🔧 ЕСЛИ БОТ ПАДАЕТ - РЕШЕНИЯ

### Проблема: "UnicodeDecodeError"
**Решение:** В Railway добавь переменную:
```env
PYTHONIOENCODING=utf-8
```

### Проблема: Бот падает каждые 30 минут
**Решение:** Railway бесплатный план имеет ограничения. Бот будет работать но может засыпать.

### Проблема: 409 Conflict Error
**Решение:** Убедись что только ОДИН инстанс бота запущен. Останови все локальные копии.

## 🎯 РЕКОМЕНДАЦИИ

### Для стабильной работы:
1. Используй **keep_alive.py** — он автоматически перезапускает бота
2. Добавь **PYTHONIOENCODING=utf-8** в переменные окружения
3. Не запускай несколько копий бота одновременно

- В Railway Dashboard можно смотреть логи
- Логи покажут почему бот упал (если упадёт)
